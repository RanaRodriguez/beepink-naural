# BeePink Naural

A web app for generating binaural beats overlaid with pink noise.

## Features
- **Pink Noise Generator:** Gentle background noise for relaxation.
- **Binaural Beats:** Configurable carrier frequency and beat frequency for various brainwave states (Delta, Theta, Alpha, Beta, Gamma).
- **Mixer:** Separate volume controls for noise and beats.
- **Modern UI:** Dark, eye-friendly design.
- **Dual Modes:**
    - **Front:** Classic Binaural Beats + Pink Noise.
    - **Back:** "Naural Pulse" (Isochronic & Spatial Pink Noise Modulation).

## Technology
- React 19
- Vite
- TypeScript
- Tailwind CSS
- Web Audio API (Native Browser Audio)

## Development
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start development server:
   ```bash
   pnpm run dev
   ```
